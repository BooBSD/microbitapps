# micro:bit apps
Simple apps for micro:bit v2

`main.py` is launcher for other apps

Requirements
------------

`pip install uflash`
`pip install microfs`

Installation
------------

First of all you have to reflash your micro:bit without any start-up script:

`uflash`

And copy launcher:

`ufs put main.py`

And some other apps, for example:

`ufs put pixels.py`
